div태그와 span의 차이점2 : 영역지정방식

div태그 : 사각형 박스로 영역을 지정함

1. https://github.com/ 로 접속하여 가입을 진행해 주세요. 2. git config --global [user.nam](http://user.name)e “본인이름” git config --global [user.](http://user.name)email “본인이메일” 가입정보에 따라서 다시 설정해 주세요. 3. 테스트를 위해 프로젝트를 새로 만들고 git init부터 텍스트 파일 생성및 수정후 커밋 2회까지 실습때처럼 순차적으로 진행하라 4. github에 접속하여 repository를 새로 생성하고 생성시 나오는 remote코드를 복사하여 터미널에서 실행시켜줘라 ex) `git remote add origin https://github.com/깃이름/레파지토리명.git` 5. git push origin main을 입력하여 원격저장소 main에 commit한 코드를 보내준다. *혹시 에러가 발생한다면 git remote -v를 이용하여 이메일이 적혀있는지 확인 git remote set-url origin https://깃허브닉네임@github.com/레파지토리명.git 을 입력후 다시 git push origin main해준다. 6. github 새로고침시 커밋데이터가 원격저장소에 반영되었을 것이다. 7. 프로젝트 내부의 텍스트 파일을 다시 수정하고 commit 후 push를 진행해서 commit데이터가 업로드되는지 확인해 보자
